# Support

If you need help with OneTaskFocus, use the support contact below.

## Contact

- Email: `support@example.com`

Replace the email above with your real support address before publishing the app.

## When Contacting Support

Include:

- device model
- iOS version
- app version
- a short description of the issue
- steps to reproduce the issue, if possible

## Response Scope

Support can help with:

- app behavior issues
- timer or notification problems
- purchase or installation questions
- general product questions
